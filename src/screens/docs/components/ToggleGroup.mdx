# ToggleGroup

並んだ toggle button group。 single (1 つだけ ON) と multiple (複数 ON) を切替えられる。 segmented control、 text formatting bar、 view switcher (List / Grid / Card) 等の rounded container で 1 セット。

## Anatomy

Root の中に Item を並べるだけのシンプル compound。 Item は data-state=on で塗りつぶし表示。

<ul className="flex flex-col gap-2 rounded-md border border-border p-4">
  {PARTS.map((p) => (
    <li key={p.name} className="grid grid-cols-[14rem_1fr] gap-3">
      <span className="font-mono text-fg text-xs">{p.name}</span>
      <span className="text-fg-muted text-xs leading-relaxed">
        {p.description}
      </span>
    </li>
  ))}
</ul>

## Single

default。 1 つだけ ON。 view switcher / sort 方向選択など。

<Example
  code={`<ToggleGroup.Root defaultValue={["list"]}>
  <ToggleGroup.Item value="list">List</ToggleGroup.Item>
  <ToggleGroup.Item value="grid">Grid</ToggleGroup.Item>
  <ToggleGroup.Item value="card">Card</ToggleGroup.Item>
</ToggleGroup.Root>`}
>
  <ToggleGroup.Root defaultValue={["list"]}>
    <ToggleGroup.Item value="list">List</ToggleGroup.Item>
    <ToggleGroup.Item value="grid">Grid</ToggleGroup.Item>
    <ToggleGroup.Item value="card">Card</ToggleGroup.Item>
  </ToggleGroup.Root>
</Example>

## Multiple

multiple で複数同時 ON 可。 text formatting bar (B / I / U) など。

<Example
  code={`<ToggleGroup.Root multiple defaultValue={["bold"]}>
  <ToggleGroup.Item value="bold"><Bold /></ToggleGroup.Item>
  <ToggleGroup.Item value="italic"><Italic /></ToggleGroup.Item>
  <ToggleGroup.Item value="underline"><Underline /></ToggleGroup.Item>
</ToggleGroup.Root>`}
>
  <ToggleGroup.Root multiple defaultValue={["bold"]}>
    <ToggleGroup.Item value="bold" aria-label="Bold">
      <Bold className="size-4" />
    </ToggleGroup.Item>
    <ToggleGroup.Item value="italic" aria-label="Italic">
      <Italic className="size-4" />
    </ToggleGroup.Item>
    <ToggleGroup.Item value="underline" aria-label="Underline">
      <Underline className="size-4" />
    </ToggleGroup.Item>
  </ToggleGroup.Root>
</Example>

## Disabled

Root に disabled で全体、 Item に disabled で個別。

<Example
  code={`<ToggleGroup.Root defaultValue={["day"]}>

  <ToggleGroup.Item value="week">Week</ToggleGroup.Item>
  <ToggleGroup.Item value="month" disabled>Month (locked)</ToggleGroup.Item>
</ToggleGroup.Root>`}
>
  <ToggleGroup.Root defaultValue={["day"]}>
    <ToggleGroup.Item value="day">Day</ToggleGroup.Item>
    <ToggleGroup.Item value="week">Week</ToggleGroup.Item>
    <ToggleGroup.Item value="month" disabled>
      Month (locked)
    </ToggleGroup.Item>
  </ToggleGroup.Root>
</Example>
