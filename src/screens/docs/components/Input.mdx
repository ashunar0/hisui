# Input

single-line の text input。 variant × size の 2 軸。 default は outline / md。 form-level の error は invalid prop で aria-invalid + danger-border に切り替わる。

## Usage

default は outline / md。

<Example code={InputUsageSource}>
  <InputUsageDemo />
</Example>

## Import

```tsx
import { Input } from "@/components/ui/input";
```

```tsx
<Input placeholder="..." />
```

## Variants

outline = 標準 / subtle = 塗り (form 内に密度) / flushed = 下線のみ (inline 編集)。

<Example code={InputVariantsSource}>
  <InputVariantsDemo />
</Example>

## Sizes

xs / sm / md / lg の 4 段階。 flushed のみ size 不問で px-3 固定。

<Example code={InputSizesSource}>
  <InputSizesDemo />
</Example>

## Invalid

invalid で aria-invalid + 各 variant の error スタイルに切替。

<Example code={InputInvalidSource}>
  <InputInvalidDemo />
</Example>

## Disabled

disabled で opacity 50% + cursor-not-allowed。

<Example code={InputDisabledSource}>
  <InputDisabledDemo />
</Example>
