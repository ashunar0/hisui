# NumberInput

数値専用の text input。 keyboard ↑↓ / Stepper button / Scrubber drag / mouse wheel で値変更。 min / max / step で範囲を制約、 formatOptions で通貨 / 単位 / 小数桁の format。

## Usage

default の vertical stepper。 Stepper は + / − を内蔵した helper。

<Example code={NumberInputBasicSource}>
  <NumberInputBasicDemo />
</Example>

## Import

```tsx
import { NumberInput } from "@/components/ui/number-input";
```

```tsx
<NumberInput.Root defaultValue="1">
  <NumberInput.Label />
  <NumberInput.Control>
    <NumberInput.Input />
    <NumberInput.Stepper />
  </NumberInput.Control>
</NumberInput.Root>
```

## Range + step

min / max / step で値の範囲と刻みを制御。 端で Stepper が自動 disabled。

<Example code={NumberInputRangeStepSource}>
  <NumberInputRangeStepDemo />
</Example>

## Cart style

− Input + を横並び。 IncrementTrigger / DecrementTrigger を Control の両端に直接配置。 EC の数量調整 UI。

<Example code={NumberInputCartSource}>
  <NumberInputCartDemo />
</Example>

## Scrubbable label

Scrubber で Label を wrap、 drag で値変更。 Figma / Photoshop の数値 input 風。 ValueText で current 表示。

<Example code={NumberInputScrubbableSource}>
  <NumberInputScrubbableDemo />
</Example>

## Currency format

formatOptions で通貨 / 単位 / 小数桁を整形。 keyboard 入力中は raw、 blur で format される。
