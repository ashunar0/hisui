# QR Code

Ark UI ベースの QR code generator。 value を渡せば SVG pattern を描画。 中央に logo を重ねたい時は Overlay と encoding.ecc=H をセットで。 Center wrapper で Frame と Overlay の center を保つのが落とし穴。

## Anatomy

Center は ui-lab 独自の relative wrapper。 Frame だけなら Center は省略可だが、 Overlay や DownloadTrigger を併用する時は必須。

<ul className="flex flex-col gap-2 rounded-md border border-border p-4">

  ))}
</ul>

## Logo overlay + download

Overlay で中央に icon を重ねる時は encoding.ecc=H で誤り訂正を最大に。 DownloadTrigger は asChild で Button に差し替えて見た目を合わせる。

<Example
  code={`<QrCode.Root
  value="otpauth://totp/ui-lab:asahi?secret=JBSWY3DPEHPK3PXP&issuer=ui-lab"
  encoding={{ ecc: "H" }}
>
  <QrCode.Center>
    <QrCode.Frame>
      <QrCode.Pattern />
    </QrCode.Frame>
    <QrCode.Overlay>
      <Sparkles className="size-4" />
    </QrCode.Overlay>
  </QrCode.Center>
  <QrCode.DownloadTrigger
    fileName="2fa-qr.png"
    mimeType="image/png"
    asChild
  >
    <Button variant="outline" size="sm">
      <Download className="size-4" /> Download PNG
    </Button>
  </QrCode.DownloadTrigger>
</QrCode.Root>`}
>
  <QrCode.Root
    value="otpauth://totp/ui-lab:asahi?secret=JBSWY3DPEHPK3PXP&issuer=ui-lab"
    encoding={{ ecc: "H" }}
  >
    <QrCode.Center>
      <QrCode.Frame>
        <QrCode.Pattern />
      </QrCode.Frame>
      <QrCode.Overlay>
        <Sparkles className="size-4 text-fg" strokeWidth={2.25} />
      </QrCode.Overlay>
    </QrCode.Center>
    <QrCode.DownloadTrigger
      fileName="2fa-qr.png"
      mimeType="image/png"
      asChild
    >
      <Button variant="outline" size="sm">
        <Download className="size-4" />
        Download PNG
      </Button>
    </QrCode.DownloadTrigger>
  </QrCode.Root>
</Example>
