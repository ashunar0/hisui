# Alert

画面内に固定で出す通知。 success / danger / warning / info の意味色 + neutral の 5 variant。 Toast (一時通知) と違い inline で居座る用途。

## Usage

default は neutral。 Title だけでも成立。

<Example code={AlertUsageSource}>
  <AlertUsageDemo />
</Example>

## Import

```tsx
import { Alert } from "@/components/ui/alert";
```

```tsx
<Alert.Root>
  <Alert.Icon />
  <Alert.Title />
  <Alert.Description />
  <Alert.Actions />
</Alert.Root>
```

## Variants

5 種。 各色に合った icon を選ぶのが定石。

<Example code={AlertVariantsSource}>
  <AlertVariantsDemo />
