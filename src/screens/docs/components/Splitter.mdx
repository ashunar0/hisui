# Splitter

resize 可能な分割 panel。 sidebar + main、 editor + preview + console、 3 column workspace 等。 panels prop で min / max を宣言、 ResizeTrigger を panel 間に挟むだけ。 親要素に明示的な高さが必須 (内部 inline style で height:100% を強制するため)。

## Usage

sidebar (15-40%) + main の 2 panel。 親に h-48 を当てて高さを与える。

<Example code={SplitterBasicSource}>
  <SplitterBasicDemo />
</Example>

## Import

```tsx
import { Splitter } from "@/components/ui/splitter";
```

```tsx
<div className="h-48">
  <Splitter.Root
    panels={[{ id: "a" }, { id: "b" }]}
    defaultSize={[50, 50]}
  >
    <Splitter.Panel id="a" />
    <Splitter.ResizeTrigger id="a:b">
      <Splitter.ResizeTriggerIndicator />
    </Splitter.ResizeTrigger>
    <Splitter.Panel id="b" />
  </Splitter.Root>
</div>
```

## 3 panels (vertical)

orientation=vertical で縦並び。 editor + preview + console の典型 layout。

## Collapsible

panel に collapsible:true を渡すと 0% まで畳める。 chat の右 detail pane や設定 inspector に。

## Anatomy

Root に panels を宣言し、 中に Panel と ResizeTrigger を交互に並べる。 trigger の id は隣接 panel を colon で繋ぐ。
