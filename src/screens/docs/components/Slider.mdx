# Slider

Track 上の Thumb を drag して数値を選ぶ。 単一値の連続入力 (volume / opacity) や、 2 つの Thumb で範囲指定 (price range)、 中央起点で正負の offset (EQ / bias) など 1 primitive で広く使える。

## Usage

uncontrolled (defaultValue) の最小例。

<Example code={SliderBasicSource}>
  <SliderBasicDemo />
</Example>

## Import

```tsx
import { Slider } from "@/components/ui/slider";
```

```tsx
<Slider.Root defaultValue={[40]}>
  <Slider.Label />
  <Slider.Control>
    <Slider.Track>
      <Slider.Range />
    </Slider.Track>
    <Slider.Thumb index={0} />
  </Slider.Control>
</Slider.Root>
```

## Controlled

value + onValueChange で外部 state と同期。 表示は ValueText で。

</Example>

## Range

Thumb を 2 つ並べて範囲指定。 minStepsBetweenThumbs で交差を防ぐ。

<Example code={SliderRangeSource}>

</Example>

## Step + markers

step で刻み、 MarkerGroup + Marker で目盛り表示。 under-value で済んだ目盛りを濃く。
