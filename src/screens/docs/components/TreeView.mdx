# TreeView

階層 list を render する primitive。 data は createTreeCollection で作って Root に collection prop で渡す。 再帰 render は NodeProvider + Branch / Item の組み合わせで自分で書く (Ark UI の規約)。 selectionMode で single / multiple、 NodeCheckbox で 3 状態 (checked / indeterminate / unchecked) も対応。

## Usage

children を持つ node = Branch、 ない node = Item。 BranchIndicator は file tree convention で chevron を rotate-90 (右→下)。

<Example code={TreeViewFileExplorerSource}>
  <TreeViewFileExplorerDemo />
</Example>

## Import

```tsx
import { TreeView } from "@/components/ui/tree-view";
import { createTreeCollection } from "@ark-ui/react/collection";
```

```tsx
const collection = createTreeCollection<Node>({
  nodeToValue: (n) => n.id,
  nodeToString: (n) => n.name,
  rootNode: { id: "ROOT", name: "", children: [...] },
});

<TreeView.Root collection={collection}>
  <TreeView.Label />
  <TreeView.Tree>
    {/* 再帰 render */}
  </TreeView.Tree>
</TreeView.Root>;
```

## Checkbox tree

selectionMode=multiple + NodeCheckbox で permission UI 風に。 親は子の選択状態に応じて自動で checked / indeterminate / unchecked を切替。

## Sidebar nav

ItemIndicator を leaf の右端に置くと、 selected な行に dot が出る。 Item を justify-between にして label と Indicator を両端配置。
