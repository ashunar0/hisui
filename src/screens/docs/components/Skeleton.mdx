# Skeleton

読み込み中の placeholder。 size / shape は className で指定する。 中身は無く、 animate-pulse の脈打つ bg-surface-sunken のみ。 1 つの primitive を組み合わせて card / row / grid の loading 状態を組む。

## Usage

単体は width / height を class で指定するだけ。

<Example code={SkeletonUsageSource}>
  <SkeletonUsageDemo />
</Example>

## Import

```tsx
import { Skeleton } from "@/components/ui/skeleton";
```

```tsx
<Skeleton className="h-4 w-40" />
```

## Shapes

rounded-full で円、 rounded-md で button / chip 風、 h-1 + w-full で text 行。

<Example code={SkeletonShapesSource}>
  <SkeletonShapesDemo />
</Example>

## Profile card

avatar + name + email + bio 2 行 + action button。 bio の width 違いで実 text の揺らぎを再現。

## Table rows

header + 5 行。 各列で width をずらして実データの不揃いを表現。

## Stat grid

Dashboard widget の loading。 数字行は h-7 大きめ、 補足行は h-3 + 60% 幅で hierarchy を残す。
