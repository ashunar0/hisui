# Separator

section / item の区切り線。 1px の bg-border。 orientation で horizontal / vertical 切替。 default は装飾 (role=none) で、 意味のある区切りなら decorative=false にして screen reader に届ける。

## Usage

default は horizontal。 縦に積んだ block の間に挟む。

<Example code={SeparatorUsageSource}>
  <SeparatorUsageDemo />
</Example>

## Import

```tsx
import { Separator } from "@/components/ui/separator";
```

```tsx
<Separator />
<Separator orientation="vertical" />
```

## Vertical

orientation=vertical で 1px の縦線。 flex-row の inline 要素間に。
