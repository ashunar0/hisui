# Button

アクションのトリガー。 variant (見た目) × colorPalette (意味の色) × size の 3 軸で組み合わせる。 Chakra UI v3 と prop 互換。

## Usage

default は solid / neutral / md。

<Example code={ButtonUsageSource}>
  <ButtonUsageDemo />
</Example>

## Import

```tsx
import { Button } from "@/components/ui/button";
```

```tsx
<Button>Save changes</Button>
```

## Variants

6 種。 solid が最も強く、 plain が最も弱い。

<Example code={ButtonVariantsSource}>
  <ButtonVariantsDemo />
</Example>

## Color palettes

意味を表す 5 色。 例は solid で比較。

  <ButtonColorPalettesDemo />

## Sizes

xs / sm / md / lg の 4 段階。

<Example code={ButtonSizesSource}>
  <ButtonSizesDemo />
</Example>

## With icon

children に lucide-react の icon を並べるだけ。 gap は size 連動。

<Example code={ButtonWithIconSource}>
  <ButtonWithIconDemo />

## Loading

loading で built-in Spinner を表示。 loadingText で文言差し替え。

  <ButtonLoadingDemo />
</Example>

## As child

asChild で Button の見た目を別要素 (a / Link 等) に移譲する。
