# Stack

flex container の薄いラッパー。 direction / gap / align / justify / wrap を semantic enum で受ける。 1 階層の flex を書くたびに class 並べるのを減らすのが目的。

## Usage

default は column / gap=md。 縦積みの最頻パターン。

<Example code={StackBasicSource}>
  <StackBasicDemo />
</Example>

## Import

```tsx
import { Stack } from "@/components/ui/stack";
```

```tsx
<Stack direction="row" gap="md" align="center" justify="between">
  ...
</Stack>
```
