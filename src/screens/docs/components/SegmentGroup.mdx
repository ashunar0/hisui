# SegmentGroup

iOS 風の segmented control。 常に 1 つだけ選択された single radio で、 選択中 item の下に浮かぶ Indicator が滑らかに追従する。 ToggleGroup と違って bordered ではなく sunken 背景、 view switcher / time period などに向く。

## Usage

time period switcher の典型。 Indicator を Item 群より前に置く。

<Example code={SegmentGroupUsageSource}>
  <SegmentGroupUsageDemo />
</Example>

## Import

```tsx
import { SegmentGroup } from "@/components/ui/segment-group";
```

```tsx
<SegmentGroup.Root>
  <SegmentGroup.Indicator />
  <SegmentGroup.Item value="...">
    <SegmentGroup.ItemText>...</SegmentGroup.ItemText>
  </SegmentGroup.Item>
</SegmentGroup.Root>
```
