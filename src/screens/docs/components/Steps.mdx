# Steps

多段 form / onboarding / wizard の進行管理。 number の Indicator + Separator + Content の組み合わせで、 horizontal の wizard も vertical の sidebar 型 form もこの 1 primitive で組める。

## Usage

3 step wizard。 Account / Profile / Confirm の Indicator + Label と、 Content × N + CompletedContent + Prev / Next。

<Example code={StepsWizardSource}>
  <StepsWizardDemo />
</Example>

## Import

```tsx
import { Steps } from "@/components/ui/steps";
```

```tsx
<Steps.Root count={N}>
  <Steps.List>
    <Steps.Item index={0}>
      <Steps.Trigger>
        <Steps.Indicator>
          <Steps.CompleteIndicator />
        </Steps.Indicator>
      </Steps.Trigger>
      <Steps.Separator />
    </Steps.Item>
  </Steps.List>
  <Steps.Content index={0} />
  <Steps.CompletedContent />
  <Steps.PrevTrigger />
  <Steps.NextTrigger />
</Steps.Root>
```

## Compact (number only)

数字だけの軽量 indicator + Context で "Step N / Total"。 sub-form の進捗表示に。

## Vertical

orientation=vertical で sidebar 配置。 左に Steps、 右に Content の 2 column。
