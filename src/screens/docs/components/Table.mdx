# Table

plain HTML table 上に variant / size / interactive / stickyHeader を Context で配る compound。 ソート用に useTableSort hook + SortableHead も同梱。 Chakra v3 と part 名互換 (Root / Header / Body / Footer / Row / Head / Cell)。

## Usage

line (default) / outline / striped の 3 variant。 layout は同じで枠と背景だけ切替。

<Example code={TableVariantsSource}>
  <TableVariantsDemo />
</Example>

## Import

```tsx
import { Table, useTableSort } from "@/components/ui/table";
```

```tsx
<Table.Root variant="outline" size="md">
  <Table.Header>
    <Table.Row>
      <Table.Head>Name</Table.Head>
    </Table.Row>
  </Table.Header>
  <Table.Body>
    <Table.Row>
      <Table.Cell>Acme</Table.Cell>
    </Table.Row>
  </Table.Body>
</Table.Root>
```

## Sizes

sm / md / lg の 3 段階。 Head height と Cell padding が連動。

## Sortable with footer

## Sticky header

stickyHeader を true にすると thead が scroll 時に固定。 親に max-h + overflow-y-auto が必要。
