# ColorPicker

2D area + channel slider + hex / rgba / hsla / hsba format 切替 + preset swatches まで揃った compound。 controlled mode は parseColor で初期化、 FormatTrigger を使うなら format も controlled + onFormatChange で sync が必須。

## Usage

Trigger (現在色の swatch) を click → popup で Area + Hue slider + Alpha slider + format 切替 + preset swatches。 controlled で format も sync。

<Example code={ColorPickerFullSource}>
  <ColorPickerFullDemo />
</Example>

## Import

```tsx
import { ColorPicker } from "@/components/ui/color-picker";
import { parseColor } from "@ark-ui/react/color-picker";
```

```tsx
<ColorPicker.Root value={color} onValueChange={...}>
  <ColorPicker.Label />
  <ColorPicker.Control>
    <ColorPicker.ChannelInput channel="hex" />
    <ColorPicker.Trigger>
      <ColorPicker.ValueSwatch />
    </ColorPicker.Trigger>
  </ColorPicker.Control>
  <ColorPicker.Positioner>
    <ColorPicker.Content>
      <ColorPicker.Area />
      <ColorPicker.ChannelSlider channel="hue" />
      <ColorPicker.SwatchGroup />
    </ColorPicker.Content>
  </ColorPicker.Positioner>
  <ColorPicker.HiddenInput />
</ColorPicker.Root>
```

## Swatches only

preset 群だけで Area / Slider 無し。 brand color picker / settings の簡易選択に。
