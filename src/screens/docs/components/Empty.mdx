# Empty

データが無い時の placeholder。 dashed border の box に Icon + Title + Description + Actions を center 寄せで並べる。 list / search / inbox の zero-state、 onboarding の初手 CTA に。

## Usage

Icon + Title + Description + 1 CTA の典型。

<Example code={EmptyUsageSource}>
  <EmptyUsageDemo />
</Example>

## Import

```tsx
import { Empty } from "@/components/ui/empty";
```

```tsx
<Empty.Root>
  <Empty.Icon />
  <Empty.Title />
  <Empty.Description />
  <Empty.Actions />
</Empty.Root>
```
