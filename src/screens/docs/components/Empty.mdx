# Empty

データが無い時の placeholder。 dashed border の box に Icon + Title + Description + Actions を center 寄せで並べる。 list / search / inbox の zero-state、 onboarding の初手 CTA に。

## Anatomy

Root の中に Icon / Title / Description / Actions を順に並べる。 全部任意 — Title だけでも成立。

<ul className="flex flex-col gap-2 rounded-md border border-border p-4">
  {PARTS.map((p) => (
    <li key={p.name} className="grid grid-cols-[14rem_1fr] gap-3">
      <span className="font-mono text-fg text-xs">{p.name}</span>
      <span className="text-fg-muted text-xs leading-relaxed">
        {p.description}
      </span>
    </li>
  ))}
</ul>

## Usage

Icon + Title + Description + 1 CTA の典型。

<Example
  code={`<Empty.Root>
  <Empty.Icon><Inbox /></Empty.Icon>
  <Empty.Title>No messages</Empty.Title>
  <Empty.Description>Your inbox is empty. New messages will show up here.</Empty.Description>
  <Empty.Actions>
    <Button>Compose</Button>
  </Empty.Actions>
</Empty.Root>`}
>
  <Empty.Root className="w-full">

      <Inbox />

    <Empty.Title>No messages</Empty.Title>

    </Empty.Description>
    <Empty.Actions>
      <Button>
        <Plus className="size-4" />
        Compose

    </Empty.Actions>
  </Empty.Root>
</Example>

## Title only

一番軽い形。 list 末尾の inline placeholder に。

<Example
  code={`<Empty.Root className="p-6">

</Empty.Root>`}
>

    <Empty.Title>No results</Empty.Title>
  </Empty.Root>
</Example>

## Search zero-state

検索結果が無い時。 query を文中に出して reset を促す。
