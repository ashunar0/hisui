# Collapsible

単一の content を開閉する primitive。 Accordion との違いは、 Accordion が複数 item の group で 1 個だけ open する仕組みなのに対し、 Collapsible は完全に独立した 1 単位の開閉。 1 つだけ畳みたい時に使う。

## Usage

Context render-prop で open に応じて Trigger ラベルを切替。 Content を Trigger の上に置くと「下にボタン」レイアウトに。

<Example code={CollapsibleUsageSource}>
  <CollapsibleUsageDemo />
</Example>

## Import

```tsx
import { Collapsible } from "@/components/ui/collapsible";
```

```tsx
<Collapsible.Root>
  <Collapsible.Trigger>
    <Collapsible.Indicator />
    ...
  </Collapsible.Trigger>
  <Collapsible.Content>...</Collapsible.Content>
</Collapsible.Root>
```

## Advanced options panel

filter panel の「上級者向け設定」のような pattern。 Trigger を上に、 Content を bordered box で下に。
