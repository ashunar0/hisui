# Toolbar

IconButton / ToggleGroup / button 等を 1 列に並べる bordered rail。 zag-js は使わず plain HTML + role=toolbar の単純 wrapper。 group の区切りには Toolbar.Separator を挟む。

## Usage

IconButton + ToggleGroup を Separator で区切る rich text editor の典型例。 a11y のために Toolbar.Root に aria-label を必ず付ける。

<Example code={ToolbarRichTextEditorSource}>
  <ToolbarRichTextEditorDemo />
</Example>

## Import

```tsx
import { Toolbar } from "@/components/ui/toolbar";
```

```tsx
<Toolbar.Root aria-label="...">
  ...buttons / ToggleGroup
  <Toolbar.Separator />
  ...
</Toolbar.Root>
```
