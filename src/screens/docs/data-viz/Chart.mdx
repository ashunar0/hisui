# Chart

recharts ベースのデータ可視化 wrapper。 ui-lab 独自の薄 wrap は Container (ResponsiveContainer + aspect) と Tooltip (token 色に揃えた cursor + content) の 2 つだけで、 中の chart 本体は recharts の AreaChart / LineChart / BarChart / PieChart 等をそのまま使う。 軸 / grid / 色は CSS 変数 (`--color-fg-muted` / `--color-border` / 等) で token と同期させる。

## Anatomy

2 part だけ。 残りは recharts の API そのまま。

    </span>
  </li>
</ul>

## Area chart

revenue 推移の定番。 stroke + fill + fillOpacity で線と塗りを分離。 axisLine / tickLine を false にして minimal 寄りに。

## Line chart (multi series)

同じ Chart.Container の中に Line を 2 本。 Tooltip は 2 entry 自動表示。 secondary は strokeDasharray で破線に。

## Bar chart

channel 別の breakdown。 radius で top-left/right だけ丸める (bar の上端を rounded)。
