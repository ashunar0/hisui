# Typography

font-family は Wix Madefor Text Variable を `--font-sans` に束ねる。 サイズ・weight・leading は Tailwind default をそのまま使い、 文字色だけ semantic token に通す方針。

## Heading scale

Heading primitive の size。 8 段階。 3xl 以上は tracking-tight、 font-weight は semibold 固定。

## Body scale

本文・UI で使うサイズ。 Tailwind default をそのまま使う。

## Weights

Wix Madefor Text Variable は 400 / 500 / 600 / 700 を主に使う。

## Foreground tokens

文字色は raw color を使わず semantic token に通す。 詳細は Colors を参照。

## Leading & tracking

長文には leading-relaxed、 大きな Heading には tracking-tight を組み合わせる。
