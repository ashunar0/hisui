# ScrollArea

custom scrollbar をかぶせるための wrapper。 native scrollbar を hide して Thumb を上書き描画するので、 OS によらず統一された見た目になる。 Root に高さを与えるのが基本。

## Usage

Root に h-N で高さを当てて、 中身が溢れた分だけ vertical scroll。

<Example code={scrollAreaVertical.code} html={scrollAreaVertical.html}>
  <ScrollAreaVerticalDemo />
</Example>

## Import

```tsx
import { ScrollArea } from "@/components/ui/scroll-area";
```

```tsx
<ScrollArea.Root>
  <ScrollArea.Viewport>
    <ScrollArea.Content>...</ScrollArea.Content>
  </ScrollArea.Viewport>
  <ScrollArea.Scrollbar orientation="vertical">
    <ScrollArea.Thumb />
  </ScrollArea.Scrollbar>
</ScrollArea.Root>
```

## Horizontal

Content を w-max + flex-row にして横に伸ばす。 Scrollbar orientation=horizontal。
