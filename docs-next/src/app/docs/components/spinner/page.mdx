# Spinner

loading 中の汎用 indicator。 Lucide Loader2 を回転 animation で wrap した SVG。 size と color は className で。 Button / IconButton の中に置けば submit pending、 単体置きで全画面 loading にも。

## Usage

単体は class 無しで size-4 の標準サイズ。

<Example code={spinnerUsage.code} html={spinnerUsage.html}>
  <SpinnerUsageDemo />
</Example>

## Import

```tsx
import { Spinner } from "@/components/ui/spinner";
```

```tsx
<Spinner />
```

## Sizes

size-\* で大きさを切り替え。 周辺の text や button に合わせる。

<Example code={spinnerSizes.code} html={spinnerSizes.html}>
  <SpinnerSizesDemo />
</Example>

## Colors

text-\* で色付け。 fg / fg-soft / fg-muted を context に合わせて。

<Example code={spinnerColors.code} html={spinnerColors.html}>
  <SpinnerColorsDemo />
</Example>

## In Button

submit pending の典型。 label と並べる時は size-4、 label を残すか差し替えるかは UI 次第。

<Example code={spinnerInButton.code} html={spinnerInButton.html}>
  <SpinnerInButtonDemo />
</Example>

## In IconButton
