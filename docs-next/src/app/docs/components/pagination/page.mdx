# Pagination

list / table の page 切替 control。 count + pageSize から totalPages を自動計算し、 siblingCount + Ellipsis で番号の表示量を絞る。 table footer の compact 版 (Prev / Next + 件数表示) も Context render-prop で組める。

## Usage

count=100 / pageSize=10 で 10 page。 siblingCount=1 で current の左右 1 つずつ + Ellipsis。

<Example code={paginationBasic.code} html={paginationBasic.html}>
  <PaginationBasicDemo />
</Example>

## Import

```tsx
import { Pagination } from "@/components/ui/pagination";
```

```tsx
<Pagination.Root count={count} pageSize={10}>
  <Pagination.PrevTrigger />
  <Pagination.Context>
    {({ pages }) =>
      pages.map((page, index) =>
        page.type === "page" ? (
          <Pagination.Item key={index} {...page}>
            {page.value}
          </Pagination.Item>
        ) : (
          <Pagination.Ellipsis key={index} index={index} />
        ),
      )
    }
  </Pagination.Context>
  <Pagination.NextTrigger />
</Pagination.Root>
```

## Sibling count

siblingCount を変えると番号の表示量が変わる。 0 で current のみ、 2 で current ± 2。
