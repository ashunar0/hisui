# Editable

inline で text を直接編集する compound。 表示 mode と edit mode を 1 場所で切替えるので、 別 modal を開かずに名前 / title / 短い設定値をさっと書き換えできる。 Notion / Linear の inline rename 風。

## Usage

Preview text を dblclick (default は focus 系) で edit mode 入り。 Enter で確定、 Esc で破棄。

<Example code={editableBasic.code} html={editableBasic.html}>
  <EditableBasicDemo />
</Example>

## Import

```tsx
import { Editable } from "@/components/ui/editable";
```

```tsx
<Editable.Root defaultValue="...">
  <Editable.Label />
  <Editable.Area>
    <Editable.Preview />
    <Editable.Input />
  </Editable.Area>
  <Editable.Control>
    <Editable.EditTrigger />
    <Editable.SubmitTrigger />
    <Editable.CancelTrigger />
  </Editable.Control>
</Editable.Root>
```

## Placeholder

defaultValue 無しなら placeholder が薄い色で出る。 click で edit mode。

<Example code={editablePlaceholder.code} html={editablePlaceholder.html}>
  <EditablePlaceholderDemo />
</Example>

## Activation mode: dblclick

activationMode=dblclick で 2 回 click したときだけ edit mode に。 誤入り防止。

<Example code={editableDblclick.code} html={editableDblclick.html}>
  <EditableDblclickDemo />
</Example>

## Submit mode

submitMode=enter で blur では確定せず、 Enter のみで commit。 form の submit 風挙動に合わせたい時。

<Example code={editableSubmitMode.code} html={editableSubmitMode.html}>
  <EditableSubmitModeDemo />
</Example>
