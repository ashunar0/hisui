# Listbox

常時展開の候補 list。 popup を開かずにそのまま選ばせる Select。 form の inline select / settings panel / sidebar nav 等に向く。 single / multiple、 filter input、 group 分けすべて対応。

## Usage

createListCollection で items を渡し、 Item に item prop で各要素を渡す。 ItemIndicator で選択 check。

<Example code={listboxBasic.code} html={listboxBasic.html}>
  <ListboxBasicDemo />
</Example>

## Import

```tsx
import { Listbox, createListCollection } from "@/components/ui/listbox";
```

```tsx
<Listbox.Root collection={collection}>
  <Listbox.Label />
  <Listbox.Content>
    <Listbox.Item item={item}>
      <Listbox.ItemText />
      <Listbox.ItemIndicator />
    </Listbox.Item>
  </Listbox.Content>
</Listbox.Root>
```

## Filterable

Input を置いて外部 state で filter。 0 件は Empty fallback。 popup じゃない Select 風 UI に。

<Example code={listboxFilterable.code} html={listboxFilterable.html}>

</Example>

## Multiple + grouped

selectionMode=multiple で複数選択、 ItemGroup + ItemGroupLabel で category 分け。

## Anatomy

Root の中に Label / Input (任意) / Content。 Content の中に Item 群 (+ Group 任意)、 Empty fallback。
