# Textarea

multi-line の text input。 Input と prop API を揃え、 min-height + 縦 resize 対応。 default は outline / md。

## Usage

default は outline / md。

<Example code={textareaUsage.code} html={textareaUsage.html}>
  <TextareaUsageDemo />
</Example>

## Import

```tsx
import { Textarea } from "@/components/ui/textarea";
```

```tsx
<Textarea placeholder="..." />
```

## Variants

Input と同じ 3 種。 form の周囲レイアウトに合わせて選ぶ。

<Example code={textareaVariants.code} html={textareaVariants.html}>
  <TextareaVariantsDemo />
</Example>

## Sizes

xs / sm / md / lg の 4 段階。 min-height が変わる。

<Example code={textareaSizes.code} html={textareaSizes.html}>
  <TextareaSizesDemo />
</Example>

## Invalid

invalid で aria-invalid + 各 variant の error スタイル。

<Example code={textareaInvalid.code} html={textareaInvalid.html}>
  <TextareaInvalidDemo />
</Example>

## Disabled

disabled で opacity 50% + cursor-not-allowed。

<Example code={textareaDisabled.code} html={textareaDisabled.html}>
  <TextareaDisabledDemo />
</Example>

## Fixed rows

rows 属性で初期高さを直接指定。 resize-y で範囲広げ可能。

<Example code={textareaFixedRows.code} html={textareaFixedRows.html}>

</Example>
