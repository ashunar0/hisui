# AspectRatio

子要素を固定比率の box に閉じ込める単純 wrapper。 CSS の aspect-ratio property を ratio prop で受けるだけ。 中身は relative box の中に絶対配置 / 普通の flow どちらも置ける。

## Usage

ratio に幅÷高さを渡す。 16:9 なら 16/9、 正方形なら 1。

<Example code={aspectRatioUsage.code} html={aspectRatioUsage.html}>
  <AspectRatioUsageDemo />
</Example>

## Import

```tsx
import { AspectRatio } from "@/components/ui/aspect-ratio";
```

```tsx
<AspectRatio ratio={16 / 9}>...</AspectRatio>
```
