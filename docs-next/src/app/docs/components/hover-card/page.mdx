# HoverCard

hover で開く preview panel。 user mention / link / item の補足情報を click せずに見せる。 Tooltip より中身を自由に組めて、 Popover と違い click 不要なので軽い browsing 向き。

## Usage

Trigger に asChild で任意の inline element (link / mention 等)、 Content に Avatar + name + bio を並べる典型。 hover で開く。

<Example code={hoverCardUsage.code} html={hoverCardUsage.html}>
  <HoverCardUsageDemo />
</Example>

## Import

```tsx
import { HoverCard } from "@/components/ui/hover-card";
```

```tsx
<HoverCard.Root>
  <HoverCard.Trigger asChild>...</HoverCard.Trigger>
  <HoverCard.Content>...</HoverCard.Content>
</HoverCard.Root>
```

## With arrow

Content に Arrow + ArrowTip を入れると向き付き。 placement 追従。

## Custom delay

openDelay / closeDelay でタイミング調整。 link rich preview は少し長めの方が誤発火しない。
