# IconButton

icon だけを置く正方形ボタン。 toolbar / nav header / list row の action で使う。 Button と同じ variant × colorPalette × size の 3 軸で、 default variant だけ `ghost` (並列に置く前提)。

## Usage

default は ghost / neutral / md。 aria-label が必須。

<Example code={iconButtonUsage.code} html={iconButtonUsage.html}>
  <IconButtonUsageDemo />
</Example>

## Import

```tsx
import { IconButton } from "@/components/ui/icon-button";
```

```tsx
<IconButton aria-label="Add">
  <Plus className="size-4" />
</IconButton>
```

## Variants

6 種。 Button と同じ map。 toolbar 並列は ghost、 単発 CTA は solid / outline。
