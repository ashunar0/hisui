# Button

アクションのトリガー。 variant (見た目) × colorPalette (意味の色) × size の 3 軸で組み合わせる。 Chakra UI v3 と prop 互換。

## Usage

default は solid / neutral / md。

<Example code={usage.code} html={usage.html}>
  <ButtonUsageDemo />
</Example>

## Import

```tsx
import { Button } from "@/components/ui/button";
```

```tsx
<Button>Save changes</Button>
```

## Variants

6 種。 solid が最も強く、 plain が最も弱い。

<Example code={variants.code} html={variants.html}>
  <ButtonVariantsDemo />
</Example>

## Color palettes

意味を表す 5 色。 例は solid で比較。

<Example code={colorPalettes.code} html={colorPalettes.html}>
  <ButtonColorPalettesDemo />
</Example>

## Sizes

xs / sm / md / lg の 4 段階。

<Example code={sizes.code} html={sizes.html}>
  <ButtonSizesDemo />
</Example>

## With icon

children に lucide-react の icon を並べるだけ。 gap は size 連動。

<Example code={withIcon.code} html={withIcon.html}>
  <ButtonWithIconDemo />
</Example>

## Loading

loading で built-in Spinner を表示。 loadingText で文言差し替え。

<Example code={loading.code} html={loading.html}>
  <ButtonLoadingDemo />
</Example>

## As child

asChild で Button の見た目を別要素 (a / Link 等) に移譲する。

<Example code={asChild.code} html={asChild.html}>
  <ButtonAsChildDemo />
</Example>
