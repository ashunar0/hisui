# Tabs

水平 / 垂直に並ぶ tab で content を切り替える。 List + Indicator + Trigger × N + Content × N の compound。 segmented control 風の見た目で Indicator が滑らかに追従する。

## Usage

uncontrolled (defaultValue) の最小例。

<Example code={tabsBasic.code} html={tabsBasic.html}>
  <TabsBasicDemo />
</Example>

## Import

```tsx
import { Tabs } from "@/components/ui/tabs";
```

```tsx
<Tabs.Root defaultValue="...">
  <Tabs.List>
    <Tabs.Indicator />
    <Tabs.Trigger value="..." />
  </Tabs.List>
  <Tabs.Content value="..." />
</Tabs.Root>
```

## Controlled

value + onValueChange で外部から制御。

<Example code={tabsControlled.code} html={tabsControlled.html}>

## Variant: line

下線型。 page 内 nav や記事の章切替に。 List / Indicator / Trigger 全部 variant=line に連動する。

<Example code={tabsLine.code} html={tabsLine.html}>
  <TabsLineDemo />
</Example>

## Disabled tab

orientation=vertical で List が縦並びに。 左 nav 風 layout に使う。
