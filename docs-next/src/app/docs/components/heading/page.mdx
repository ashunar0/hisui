# Heading

見出し。 font-weight は semibold 固定、 size と as を独立に持つのが特徴。 page title は `as="h1"` + `size="2xl"` のように見た目と semantics を分けて指定する。

## Usage

default は as=h2 / size=md。 何も指定しなければ section 見出し相当。

<Example code={headingUsage.code} html={headingUsage.html}>
  <HeadingUsageDemo />
</Example>

## Import

```tsx
import { Heading } from "@/components/ui/heading";
```

```tsx
<Heading as="h1" size="2xl">Title</Heading>
```

## Sizes

8 段階。 3xl 以上は自動で tracking-tight が付く。

<Example code={headingSizes.code} html={headingSizes.html}>
  <HeadingSizesDemo />
</Example>

## As (semantic tag)

size と独立に HTML tag を選ぶ。 a11y / SEO の文書構造はここで決める。

<Example code={headingAsTag.code} html={headingAsTag.html}>
  <HeadingAsTagDemo />
</Example>

## Page title pattern

page の H1 は size=2xl + as=h1。 hero では 3xl 以上を使う。

<Example code={headingPageTitle.code} html={headingPageTitle.html}>
  <HeadingPageTitleDemo />
</Example>

## Section label pattern

docs の section 見出しは as=h2 + size=sm + text-fg-soft で控えめに。

<Example code={headingSectionLabel.code} html={headingSectionLabel.html}>
