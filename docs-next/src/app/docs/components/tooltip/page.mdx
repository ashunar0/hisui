# Tooltip

IconButton / Avatar など label 無しの UI に hover / focus で説明文を出す。 bg-fg / text-bg の暗い吹き出し + 矢印。 Portal で z-order を確実に上に。

## Usage

Trigger を IconButton に被せる典型例。 hover / focus で表示。

<Example code={tooltipBasic.code} html={tooltipBasic.html}>
  <TooltipBasicDemo />
</Example>

## Import

```tsx
import { Tooltip } from "@/components/ui/tooltip";
```

```tsx
<Tooltip.Root>
  <Tooltip.Trigger asChild>
    <IconButton />
  </Tooltip.Trigger>
  <Tooltip.Content>...</Tooltip.Content>
</Tooltip.Root>
```

## With arrow

Arrow + ArrowTip を Content の中に置くと矢印が付く。 placement に追従して向き自動。

<Example code={tooltipWithArrow.code} html={tooltipWithArrow.html}>
  <TooltipWithArrowDemo />
</Example>

## Placements

positioning.placement で 4 方向。 余白が無いと自動で flip する。

<Example code={tooltipPlacements.code} html={tooltipPlacements.html}>
  <TooltipPlacementsDemo />
</Example>

## Delay

openDelay を 0 にすると即時表示、 800 にすると遅延を強める。
