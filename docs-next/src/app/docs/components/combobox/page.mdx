# Combobox

text 入力で候補を絞り込んで選ぶ Select の上位互換。 useListCollection で項目を持ち、 タイプ毎に filter を回す。 single / multiple、 flat / group、 任意の Empty fallback まで 1 primitive で。

## Usage

useListCollection で collection と filter を作り、 onInputValueChange から filter を呼ぶ。 ItemIndicator で選択チェック。

<Example code={comboboxBasic.code} html={comboboxBasic.html}>
  <ComboboxBasicDemo />
</Example>

## Import

```tsx
import { Combobox, useListCollection } from "@/components/ui/combobox";
```

```tsx
<Combobox.Root collection={collection} onInputValueChange={...}>
  <Combobox.Label />
  <Combobox.Control>
    <Combobox.Input />
    <Combobox.Trigger />
  </Combobox.Control>
  <Combobox.Content>
    <Combobox.Empty />
    <Combobox.Item item={item}>
      <Combobox.ItemText />
      <Combobox.ItemIndicator />
    </Combobox.Item>
  </Combobox.Content>
</Combobox.Root>
```

## Grouped

ItemGroup + ItemGroupLabel で候補を category 別に。 ClearTrigger で × もつけて入力一括 clear。

## Multiple

multiple で複数選択。 選んでも Content は閉じない。 選択数は Combobox.Context render-prop で取得して Label 横に出すと UI 親切。
