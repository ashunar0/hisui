# Clipboard

clipboard API を使った copy 動作を内蔵した primitive。 Trigger を押すと value がコピーされ、 Indicator が一定時間 Check icon に切替わる。 Input / ValueText を組み替えれば install command / share URL / code snippet どのレイアウトにも乗せられる。

## Usage

Label + Input + Trigger の定番 layout。 Input readonly で編集不可に。 install command の典型。

<Example code={clipboardInstallCommand.code} html={clipboardInstallCommand.html}>
  <ClipboardInstallCommandDemo />
</Example>

## Import

```tsx
import { Clipboard } from "@/components/ui/clipboard";
```

```tsx
<Clipboard.Root defaultValue="...">
  <Clipboard.Label />
  <Clipboard.Control>
    <Clipboard.Input readOnly />
    <Clipboard.Trigger>
      <Clipboard.Indicator />
    </Clipboard.Trigger>
  </Clipboard.Control>
</Clipboard.Root>
```

## Inline share URL

pill 風に丸めて URL の右端に小さな copy button を貼る。 share UI に便利。

<Example code={clipboardInlineShareUrl.code} html={clipboardInlineShareUrl.html}>
