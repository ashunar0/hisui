# ToggleGroup

並んだ toggle button group。 single (1 つだけ ON) と multiple (複数 ON) を切替えられる。 segmented control、 text formatting bar、 view switcher (List / Grid / Card) 等の rounded container で 1 セット。

## Usage

default は single mode。 1 つだけ ON。 view switcher / sort 方向選択など。

<Example code={toggleGroupSingle.code} html={toggleGroupSingle.html}>
  <ToggleGroupSingleDemo />
</Example>

## Import

```tsx
import { ToggleGroup } from "@/components/ui/toggle-group";
```

```tsx
<ToggleGroup.Root>
  <ToggleGroup.Item value="...">...</ToggleGroup.Item>
</ToggleGroup.Root>
```

## Multiple

multiple で複数同時 ON 可。 text formatting bar (B / I / U) など。

<Example code={toggleGroupMultiple.code} html={toggleGroupMultiple.html}>
  <ToggleGroupMultipleDemo />
</Example>

## Icon + text

text alignment 等、 icon と label を並べる。 segmented control 感。
