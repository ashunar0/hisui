# Alert

画面内に固定で出す通知。 success / danger / warning / info の意味色 + neutral の 5 variant。 Toast (一時通知) と違い inline で居座る用途。

## Usage

default は neutral。 Title だけでも成立。

<Example code={alertUsage.code} html={alertUsage.html}>
  <AlertUsageDemo />
</Example>

## Import

```tsx
import { Alert } from "@/components/ui/alert";
```

```tsx
<Alert.Root>
  <Alert.Icon />
  <Alert.Title />
  <Alert.Description />
  <Alert.Actions />
</Alert.Root>
```

## Variants

5 種。 各色に合った icon を選ぶのが定石。

<Example code={alertVariants.code} html={alertVariants.html}>
  <AlertVariantsDemo />
</Example>

## Title only

Description を省くとコンパクトな 1 行 banner に。

## With actions

Alert.Actions に Button を並べる。 確認 / dismiss の 2 button が典型。
