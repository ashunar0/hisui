# Switch

on / off の 2 状態を持つ form control。 Checkbox と機能は同じだが、 即時反映する設定 (notification、 dark mode、 feature toggle) に使う。

## Usage

uncontrolled (defaultChecked) の最小例。

<Example code={switchUsage.code} html={switchUsage.html}>
  <SwitchUsageDemo />
</Example>

## Import

```tsx
import { Switch } from "@/components/ui/switch";
```

```tsx
<Switch.Root>
  <Switch.Control>
    <Switch.Thumb />
  </Switch.Control>
  <Switch.Label>...</Switch.Label>
</Switch.Root>
```

## Controlled

checked + onCheckedChange で値を制御。
