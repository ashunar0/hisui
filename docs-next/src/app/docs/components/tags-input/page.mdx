# TagsInput

chip 列 + 末尾 Input で複数 tag を入力する form primitive。 Enter で確定、 Backspace で末尾削除、 paste で delimiter 分割。 各 chip は double-click で編集モードに切替。 max / validate で filter も組める。

## Usage

Enter で追加、 chip の × で削除、 ClearTrigger で全消し。 Context render-prop で value を回して Item を map するのが基本契約。

<Example code={tagsInputSkills.code} html={tagsInputSkills.html}>
  <TagsInputSkillsDemo />
</Example>

## Import

```tsx
import { TagsInput } from "@/components/ui/tags-input";
```

```tsx
<TagsInput.Root>
  <TagsInput.Label />
  <TagsInput.Control>
    <TagsInput.Context>{renderItems}</TagsInput.Context>
    <TagsInput.Input />
  </TagsInput.Control>
</TagsInput.Root>
```
