# List

native の ul / ol を装飾する primitive。 marker variant で bullet / decimal、 plain variant で marker を消して Indicator slot に icon を入れる用途に切り替わる。 prose の中の箇条書きから check list / feature list まで一枚で。

## Usage

default は ul + marker。 children に Item を並べるだけ。

<Example code={listUsage.code} html={listUsage.html}>
  <ListUsageDemo />
</Example>

## Import

```tsx
import { List } from "@/components/ui/list";
```

```tsx
<List.Root>
  <List.Item>...</List.Item>
</List.Root>
```

## Ordered

as=ol で番号付きリストに。 marker は decimal。

<Example code={listOrdered.code} html={listOrdered.html}>
  <ListOrderedDemo />
</Example>

## With indicator

variant=plain で native marker を消し、 Item の先頭に Indicator で icon を置く。 feature / check list に。

<Example code={listWithIndicator.code} html={listWithIndicator.html}>
  <ListWithIndicatorDemo />
</Example>
