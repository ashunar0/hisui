# Card

compound primitive。 Root に variant / size を渡し、 中身は Header / Title / Description / Body / Footer を必要分だけ組み合わせる。 size は Context で全 part に流れる。

## Usage

default は outline / md。 Header + Body + Footer の典型 layout。

<Example code={cardUsage.code} html={cardUsage.html}>
  <CardUsageDemo />
</Example>

## Import

```tsx
import { Card } from "@/components/ui/card";
```

```tsx
<Card.Root>
  <Card.Header />
  <Card.Body />
  <Card.Footer />
</Card.Root>
```

## Variants

3 種。 elevated = 多層 shadow で浮き、 outline = border のみ、 subtle = 背景塗り。

<Example code={cardVariants.code} html={cardVariants.html}>
  <CardVariantsDemo />
</Example>

## Sizes

sm / md / lg の 3 段階。 Header / Body / Footer の padding が連動。

<Example code={cardSizes.code} html={cardSizes.html}>
  <CardSizesDemo />
</Example>

## Header only

Title + Description だけで成立。 Body / Footer は不要なら省く。

<Example code={cardHeaderOnly.code} html={cardHeaderOnly.html}>
  <CardHeaderOnlyDemo />
</Example>

## As child (clickable card)

asChild で Root を Link や button に差し替えると hover / focus が乗る。

<Example code={cardAsChild.code} html={cardAsChild.html}>
  <CardAsChildDemo />
</Example>

## Anatomy

どの part も任意。 最小は Root だけでも成立する。
