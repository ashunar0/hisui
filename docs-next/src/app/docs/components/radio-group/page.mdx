# RadioGroup

排他選択 (択一) の form control。 Checkbox との違いは「複数選べる / 選べない」。 Root に value を持たせ、 各 Item に value を渡す。

## Usage

uncontrolled (defaultValue) の最小例。

<Example code={radioGroupBasic.code} html={radioGroupBasic.html}>
  <RadioGroupBasicDemo />
</Example>

## Import

```tsx
import { RadioGroup } from "@/components/ui/radio-group";
```

```tsx
<RadioGroup.Root defaultValue="...">
  <RadioGroup.Item value="...">
    <RadioGroup.ItemControl />
    <RadioGroup.ItemText />
  </RadioGroup.Item>
</RadioGroup.Root>
```

## Controlled

value + onValueChange を渡して制御。 onValueChange は details.value で string を受け取る。

<Example code={radioGroupControlled.code} html={radioGroupControlled.html}>
  <RadioGroupControlledDemo />
</Example>
