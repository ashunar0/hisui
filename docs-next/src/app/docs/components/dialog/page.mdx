# Dialog

中央 modal。 Title / Description / 任意の中身 / 確定キャンセル button を組み合わせる。 Portal + Backdrop + Positioner は Content 内部で自動 mount。 multi-layer shadow + rounded-lg の floating surface 標準。

## Usage

Trigger (asChild で Button) + Title + Description + Footer の最小 modal。

<Example code={dialogUsage.code} html={dialogUsage.html}>
  <DialogUsageDemo />
</Example>

## Import

```tsx
import { Dialog } from "@/components/ui/dialog";
```

```tsx
<Dialog.Root>
  <Dialog.Trigger asChild>
    <Button>...</Button>
  </Dialog.Trigger>
  <Dialog.Content>
    <Dialog.Title />
    <Dialog.Description />
    <Dialog.CloseTrigger asChild>
      <Button />
    </Dialog.CloseTrigger>
  </Dialog.Content>
</Dialog.Root>
```

## With form

中身は任意。 Input を並べて確定 / キャンセル の典型 form modal。
