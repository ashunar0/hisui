# DatePicker

日付選択用の compound。 calendar grid (day / month / year の 3 view 切替)、 Input への直接入力、 range / multiple、 preset jump まで全部入り。 popup は Portal で escape する。

## Usage

Trigger click で calendar 開く。 day 選ぶと Input 反映、 popup 上の月名 click で month/year view にジャンプ。

<Example code={datePickerBasic.code} html={datePickerBasic.html}>
  <DatePickerBasicDemo />
</Example>

## Import

```tsx
import { DatePicker } from "@/components/ui/date-picker";
```

```tsx
<DatePicker.Root>
  <DatePicker.Label />
  <DatePicker.Control>
    <DatePicker.Input />
    <DatePicker.Trigger />
  </DatePicker.Control>
  <DatePicker.Content>
    <DatePicker.Calendar />
  </DatePicker.Content>
</DatePicker.Root>
```

## Range

selectionMode=range で 2 つの日を選び期間に。 booking / 期間 filter。

<Example code={datePickerRange.code} html={datePickerRange.html}>
  <DatePickerRangeDemo />
</Example>

## Locale (Japanese)

locale prop で曜日名 / 週先頭 / 月名を切替。

<Example code={datePickerLocale.code} html={datePickerLocale.html}>

## Fixed weeks

fixedWeeks で常に 6 週表示。 calendar の高さが月ごとに変わらなくなる (UI ガクガク防止)。

<Example code={datePickerFixedWeeks.code} html={datePickerFixedWeeks.html}>
  <DatePickerFixedWeeksDemo />
</Example>

## Anatomy

Root の中に Label / Control (Input + Trigger) / Content (Calendar)。 Calendar は DayView + MonthView + YearView を含む helper。
