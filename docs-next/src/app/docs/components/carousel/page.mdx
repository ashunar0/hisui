# Carousel

Ark UI ベースの slider。 slide / dot / Prev/Next / Autoplay / Progress を全部 dot-namespace で持つ。 Root に slideCount を渡し、 Item の index と合わせるのが基本契約。

## Usage

最小構成 + dot indicator。 Item に index、 Root に slideCount を渡す。

<Example code={carouselHero.code} html={carouselHero.html}>
  <CarouselHeroDemo />
</Example>

## Import

```tsx
import { Carousel } from "@/components/ui/carousel";
```

```tsx
<Carousel.Root slideCount={n}>
  <Carousel.ItemGroup>
    <Carousel.Item index={0} />
  </Carousel.ItemGroup>
  <Carousel.Control>
    <Carousel.PrevTrigger />
    <Carousel.NextTrigger />
  </Carousel.Control>
  <Carousel.IndicatorGroup>
    <Carousel.Indicator index={0} />
  </Carousel.IndicatorGroup>
</Carousel.Root>
```

## With progress counter

ProgressText を header に出すと「3 / 4」が見える。 IndicatorGroup を省くパターン。

## Autoplay

autoplay に delay を渡すと自動再生。 loop と組み合わせて無限ループに。 AutoplayTrigger で一時停止 toggle を出せる。
