# Checkbox

on / off / indeterminate の 3 状態を持つ form control。 Ark UI ベース。 単独 checkbox は Root + Control + Label の 3 部、 複数同時管理は Group で囲んで value array に流す。

## Usage

uncontrolled (defaultChecked) の最小例。 Label クリックでも toggle。

<Example code={checkboxUsage.code} html={checkboxUsage.html}>
  <CheckboxUsageDemo />
</Example>

## Import

```tsx
import { Checkbox } from "@/components/ui/checkbox";
```

```tsx
<Checkbox.Root>
  <Checkbox.Control />
  <Checkbox.Label>...</Checkbox.Label>
</Checkbox.Root>
```

## Controlled

checked + onCheckedChange を渡して値を制御。

<Example code={checkboxControlled.code} html={checkboxControlled.html}>
  <CheckboxControlledDemo />
</Example>

## Indeterminate

'indeterminate' で 中間状態。 子 checkbox の一部だけ選択中の親 checkbox に使う。

<Example code={checkboxIndeterminate.code} html={checkboxIndeterminate.html}>
  <CheckboxIndeterminateDemo />
</Example>

## Disabled

disabled で操作不可 + opacity-50。

<Example code={checkboxDisabled.code} html={checkboxDisabled.html}>
  <CheckboxDisabledDemo />

## Group

Checkbox.Group で value array を一括管理。 各 Root に value を渡す。
